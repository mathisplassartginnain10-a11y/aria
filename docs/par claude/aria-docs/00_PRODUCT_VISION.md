# 00 — ARIA Product Vision

## Ce qu'est ARIA

ARIA n'est pas un chatbot. ARIA n'est pas un dashboard SaaS.

ARIA est un **système d'exploitation IA personnel** qui tourne entièrement en local sur la machine de l'utilisateur. Quand l'utilisateur ouvre ARIA, il doit avoir l'impression qu'une intelligence tourne sur sa machine.

## Inspirations visuelles

- JARVIS (Iron Man) — interface holographique, anneaux rotatifs, HUD
- Apple Vision OS — Liquid Glass, depth layers, réfraction
- iOS 26 Dynamic Island — états réactifs, animations physiques
- Apple Music — glassmorphism premium
- Notion AI — fluidité conversationnelle

## Ce qu'ARIA ne doit PAS ressembler à

- Un dashboard SaaS (pas de tableaux de bord corporate)
- Une copie de ChatGPT (pas de bulles basiques sur fond blanc)
- Une app Electron générique (pas de chrome Windows visible)

## Principes de design

1. **Local First** — aucune donnée ne quitte la machine par défaut
2. **Réactivité** — chaque interaction a un feedback visuel immédiat
3. **Personnalisation** — tout est configurable (nom, icône, modèle, thème)
4. **Élégance** — qualité Apple, pas qualité app open source
5. **Performance** — 120fps, animations physiques, aucun lag perceptible
